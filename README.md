# digilab.ai Special Instagram Posts

Spezielle PNG-Generator-App fuer DigiLab.ai Instagram Posts.

Enthaelt:

- Cover-Template im Stil der gelieferten Workshop-Grafik
- News-Template fuer Titel plus Kurztext
- Terminlisten-Template mit mehreren Eintraegen
- Instagram Formate fuer Quadrat, Portrait, Story/Reel und Landscape
- DigiLab-Farbpresets plus freier Farbwähler
- eingebaute DigiLab-Logos und PNG-Export

Start:

```bash
npm install
npm run dev
```
